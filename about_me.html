Cordispeak: A Collaborative Language Translation Project

A Bridge Between Cultures

In collaboration with Chielo Depao, a fellow student at Benguet State University pursuing a degree in Tourism, we embarked on an exciting project to develop a language translation tool named Cordispeak. This innovative tool aims to facilitate seamless communication between locals and tourists in the Cordillera region, fostering cultural exchange and understanding.

Our Focus: The User Interface

As part of this collaborative effort, we concentrated on designing the user interface (UI) of Cordispeak. The UI is the visual aspect of the application that users interact with. Our goal was to create a clean, intuitive, and visually appealing interface that would make the translation process as smooth as possible.
